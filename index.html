AirBnB Data Design Project

Persona: Leah

relevant characteristics:

  • widowed, single, divorced, retired
  • home paid for
  • all kids grown up and moved out
  • extra bedrooms / baths
  • plenty of time available
  • tablet user or home computer
  • travels locally to visit kids and community events

Use Cases

Goal:

extra cash & enjoy the interaction with guests, provide nice meals, social interaction

Sign-up for an airbnb hosting account so Leah can add her listing(s):

  1. name
  2. email
  3. password
  4. age

Create a hosting profile

  1. type of house or room or ??
  2. location
  3. price
  4. dates_available

Goal:

Host wants to view their upcoming reservations

Steps to accomplish this:

  1. login to their account.
  2. click the dropdown arrow to the right of their profile picture in top right hand corner of their home page.
  3. From the dropdown list, select “Your Reservations”.

Goal:

A user needs to find out how to cancel an existing booking.

Steps to accomplish this:

  1. Login into their airbnb hosting account
  2. Click on the dropdown arrow beside “help” in upper right hand corner.
  3. From the dropdown menu, select “How do I cancel a reservation as a host?

DROP TABLE IF EXISTS reservation;

DROP TABLE IF EXISTS listing;

DROP TABLE IF EXISTS member;

CREATE TABLE member (

userId INT UNSIGNED AUTO_INCREMENT NOT NULL,

loginName VARCHAR(20)

firstName VARCHAR(30) NOT NULL,

lastName VARCHAR(30) NOT NULL,

createDate DATE

email VARCHAR(50) NOT NULL,

password CHAR(255) NOT NULL ,

>birthday DATETIME NOT NULL,

PRIMARY KEY(userId)

);

CREATE TABLE listing (

listingID INT UNSIGNED AUTO_INCREMENT NOT NULL,

listingType VARCHAR(60) NOT NULL,

street VARCHAR(50) NOT NULL,

city VARCHAR(50),

state CHAR(2) NOT

zip VARCHAR(10) NOT NULL

price DECIMAL(6,0) NOT NULL,

>datesAvailable DATETIME,

description VARCHAR(255),

image VARCHAR(15)

phone VARCHAR(15),

PRIMARY KEY(listingId)

);

CREATE TABLE reservation (

userId,

listingID,

dateWanted DATETIME NOT NULL,

PRIMARY KEY(reservationId)

);
